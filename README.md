# Bismillah Khan — Frontend Developer Portfolio

## 🌐 Live Demo

[View Portfolio](https://bismillahkhan.vercel.app)

## 👨‍💻 About

Personal portfolio website showcasing my skills, services, and projects as a Frontend and WordPress Developer based in Islamabad, Pakistan.

## ✨ Features

- Fully responsive on all devices
- Smooth animations and scroll effects
- Interactive project filtering
- Working contact form via Formspree
- Fast loading and optimized

## 🛠️ Built With

- React.js
- Tailwind CSS
- Vite
- Formspree
- Vercel

## 🚀 Getting Started

```bash
git clone https://github.com/bismillahkhanweb/portfolio.git
cd portfolio
npm install
npm run dev
```
